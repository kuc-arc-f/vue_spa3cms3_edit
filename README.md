### Summary

IndexedDB + Dexie.js + Vue CLI , cms edit app


***
### Setup

* npm install -g @vue/cli

* vue create my-project

* cd my-project

* start server

***
### npm

npm install --save dexie

***
### start server
npm run serve

* build project, deploy

npm run build
